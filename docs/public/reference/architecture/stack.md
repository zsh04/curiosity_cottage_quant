**Hardware Performance Matrix**:

| Component | Apple M2/M3 (MPS) | NVIDIA GPU (CUDA) | CPU Only (Legacy) |
| :--- | :--- | :--- | :--- |
| **Chronos Forecast** | ~80ms | ~40ms | **~1200ms (FAIL)** |

> [!CAUTION]
> **Minimum Viable Hardware**: The Cortex Engine requires **Apple Silicon (M1 Pro+)** or **NVIDIA GPU (8GB+)**. Running on CPU is considered a "Soft Failure" state due to signal latency exceeding market relevancy.
